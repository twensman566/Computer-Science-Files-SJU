Lab 7: Making simple animated simulations—animated boxes, turtles and hares

Notes:

  1. Work with your pair partner(s) on this lab but swap roles.  If you were not the recorder last time, your role will be as recorder this time.
  2. Remember that all programs are expected to contain comments for public classes, variables, constants and methods.

  4. Remember to share your files with your partner.

Part 1: Animating the turtle and the hare

In this part of the lab we're going to redo the turtle and hare race so you don't have to continually click the mouse to run the race. We will now have classes called ActiveTurtle and ActiveHare that will run the race autonomously once they are started. The ActiveTurtle and ActiveHare classes will extend the Animator class, which allows objects to act independently of each other by running in separate threads. The NewRace class will create a turtle and a hare, put them on the race course and start them; the two runners will run their race until one of them reaches the finish line. Each runner will time him/herself (fairly!) and post her/his finish time on the canvas.

Make one new Java file Hare.java and copy the contents of the file Turtle.java into it. Modify the files Turtle.java and Hare.java so they implement the Turtle and Hare classes, respectively; use your code from Lab 2 to make your classes draw the two animals. The ActiveHare and ActiveTurtle classes have instance variables of types Hare and Turtle, respectively. The ActiveTurtle class is completed for you; you should complete the ActiveHare class using the ActiveTurtle as a model.

Part 2: Adding another racer

Make a class for a new animal (not a turtle or hare) and a new corresponding active animal class. Modify the NewRace class so that there are three racers, including one of the new animal.

As usual, when you are done with the in-lab exercises above, have your lab instructor or TA check over your work before you leave the lab.
