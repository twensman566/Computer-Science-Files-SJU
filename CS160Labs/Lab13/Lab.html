Lab 13: Manipulating Text—Pig Latin

Perhaps you remember trying to speak in “Pig Latin” as a child. Words starting with consonants (all of the consonants before the first vowel) have the starting consonants removed from the front of a word and added to the end of the word followed by “ay” so that, for example, “way” becomes “ayway”, “java” becomes “avajay”, “theme” becomes “emethay", and so on. Words starting with a vowel stay the same with a “way” appended to their end so that “elated” becomes “elatedway”, “after” becomes “afterway” etc. In particular, if you had “awesome Pig Latin” it would become “awesomeway igPay atinLay”. If you need to refresh your memory more on Pig Latin, check out this website: English to Pig Latin Translator.

Part 1: A method that translates one word into Pig Latin

You will be completing a program that can take any string of zero or more words and convert it to a string of equivalent words in Pig Latin. You have been given a complete main program, GraphicsPigLatin that provides the user interface for the converter and a PigLatin class that contains static methods for converting English to Pig Latin. The user interface uses Java GUI components that we haven't studied yet, but you won't need to work with those parts of the program. The only part you need to complete is the convertWord() method in the PigLatin class. The rules for how the method should work are in the class comment at the top of the file.

Show your work to the TA or lab instructor before you go on.

Part 2: Correcting the capitalization

The convertWord() method doesn't work correctly for words beginning with a capital consonant. Modify this method so that it handles this case correctly; for example, “Pig Latin” will become “Igpay Atinlay” instead of “igPay atinLay.” The Character and String classes should provide methods appropriate for accomplishing this task.

Show your work to the TA or lab instructor before you go on.

Part 3: Translating text files into Pig Latin

Write a new Program that opens a Scanner on a text file (using a FileChooser), reads and converts each line of the file to Pig Latin, and writes each converted line to another file using a PrintWriter (also opened with a FileChooser.) Use the same methods in the PigLatin class for this program.
