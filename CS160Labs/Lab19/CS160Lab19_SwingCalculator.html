Lab 15: Using Swing Components—a Calculator

Part 1: A Simple Decimal Calculator

You have been give the start of a simple decimal calculator program, Calculator.java. Compile and run this program, experimenting with it until you understand how it works. Note that the program doesn't have a pull-down menu or “Quit” button, so you'll have to stop it manually, even if you close the window.

In this part you will finish constructing the simple calculator. Modify the Calculator.java program to create a simple general purpose calculator that operates on two fields (First and Second) containing numeric values and puts the result in the Answer field.

Follow the procedure suggested in the comments in file to add all the additional functionality. A wise approach would be to create the button panel, add the ‘+’ button to it, and add that panel to the content pane (Swing-speak for canvas), instead of the button. Make sure the program works with this change before you go on.

Now add a second button to the button panel, and add its functionality to the ActionPerformed() method. Once you have the second button working, go ahead and add the rest of the functions and check your work.

For functions requiring two numbers, like ‘+’, ‘−’, ‘×’ and ‘÷’, both the First and Second fields are used; for functions requiring one number like “log” and “tan”, the operator works on the value in the First field and displays result in the Answer field. “+/− First” changes the sign of the number in the First field; other functions that specify one or more fields work similarly. Use the Math class for log and trig functions.

Finally, add a “Quit” button that stops your program by executing the statement System.exit(0);.

Note: There are several parts of this program that use Java features we haven't studied very much yet. Just leave them as they are, and your program should work fine.
