Pre-lab exercise: Drawing randomly-colored lines

The file RandomColor.java contains a program that generates green lines on the canvas when the user drags the mouse around the canvas. Open the file in DrJava. Compile and run the program, comparing the code with the way the program works. When you are satisfied that you understand how the program does what it does, begin working on modifying the program.

Note: If you get stuck and need to start over, you can make a new copy of the original program instead of trying to unwind all the changes you made.

You are to modify the program so that instead of drawing only green line, the color for each line is selected randomly from red, green, blue, black, and yellow. You can do this by using the random number generator as in Lab 3 to choose a number between 0.0 and 1.0 and then using an if-statement to set the color based on the number that was generated: 0.0–0.2 for red, 0.2–0.4 for green, etc.). This selection statement could look something like this: 

    double selection = Math.random();
    if (selection <= 0.2)
    {
      currentColor = Color.RED;
    }
    else if (selection <= 0.4)
    {
      currentColor = Color.GREEN;
    }
    ⋮

You'll need two more else if-clauses and an else-clause
