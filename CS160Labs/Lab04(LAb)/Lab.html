Part 1: A laundry-sorting simulation

Open the file LaundrySorter.java in DrJava.

As college students, it is important to learn the proper way to sort laundry. In this part of the lab, you will write a Java program that simulates sorting laundry. The initial display should display a rectangle—the hamper—that is white, black, or red, representing a white, dark, or colored item of clothing at the top of the hamper. Next to this rectangle should be three rectangles that represent three tubs of laundry, Whites, Darks, and Colors and have appropriate colors and text labels. The user is to look at the color of the “clothing item” (colored rectangle) in the hamper and click on the appropriate tub. Each time the user correctly selects a tub, the item of clothing will be randomly replaced by another black, white, or other-colored item. If the user selects the wrong tub, the program should display an error message, and the clothing item should be unchanged; the error message should be hidden when the user selects a tub correctly.

Part 2: Keeping count of events

Change the laundry program so that it counts and displays the number of correct and incorrect choices the user has made. Your can do this by adding two int instance variables correct and wrong, both initialized to 0, adding the statement this.correct++; wherever you detect a correct choice and adding the statement this.wrong++; wherever you detect an incorrect choice. Use GLabels on the canvas to display the current values of correct and wrong; use the setLabel() method to change the text value of the label.

Show your finished program to the lab instructor or TA.
