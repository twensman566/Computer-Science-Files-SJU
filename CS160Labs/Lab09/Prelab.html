Pre-lab exercise: a new Bullseye

Make a new version of the Bullseye class from Lab 5. Use the SquareBoxes class as your basic model, and refer to the old version of the Bullseye class for specific code to place the circles. You should have an array instance variable to hold all the circles, with the outermost circle at index 0. The constructor should take a parameter int numRings that represents the total number of rings in the bullseye, counting the center circle as a ring. The constructor allocates the array to the specified size and then makes the circles with alternating default colors. Add a setColor(boolean outer, Color newColor) method that changes the color of alternate rings to the specified new color; if the parameter outer is true, it changes the outermost ring and every second ring inward; if outer is false it changes the second ring and every second ring inward.

Use the Lab 5 test program as a model to test your new Bullseye class. Be sure to test several sizes of bullseyes, including both even and odd numbers of rings, and for the color change with both even- and odd-numbered bullseyes.
